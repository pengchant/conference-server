#  党政会议管理系统

*   会议预约管理系统    (1 week)
*   会议签到管理系统    (1 week)
*   会议互动管理系统    (1 week)
*   会议记录管理系统    (1 week)
*   会议档案调阅管理系统 (1 week)
*   会议配置管理模块    (1 week)


##  后端采用技术

*   SpringBoot + MyBatis 
*   MySQL
*   Redis
*   WebSocket
*   JWT + Shiro + Swagger
*   Kafka消息队列
*   MongoDB GridFS
*   nginx + Tomcat

##  前端技术栈
*   Vue + Vuex + VueRouter
*   npm + webpack


##  项目文件目录说明

/doc    本项目过程文档

/src    后端程序源代码

conferencem.sql 本项目的mysql数据库脚本

说明：使用idea导入项目后，运行sql脚本(conferencem.sql)，需要修改自己的数据库用户名和密码。

项目启动后访问：http://localhost:9999/swagger-ui.html#/

后端所有的api都罗列于此。
